# 🥕 Karrot Market 미국 현지화 전략 프로젝트

본 프로젝트는 지역 기반 중고거래 플랫폼 **Karrot Market(당근마켓)**의 글로벌 확장 전략을 분석하고, 미국 시장 진출을 위한 **현지화 전략(Localization Strategy)**을 제안한 팀 프로젝트입니다.

특히 기존 진출 지역인 **뉴욕·뉴저지·시카고**에 이어 **캘리포니아 시장 확장 전략**을 중심으로 경쟁 플랫폼 분석, 법적 규제 조사, 서비스 기능 기획 등을 수행했습니다.

🐰 본 프로젝트는 **기업메이커 프로그램 팀 공모전 프로젝트**로 진행되었습니다.

# 🥕 Karrot Market SWOT 분석

### Strength (강점)

- 지역 기반 커뮤니티 중심 서비스
- 높은 사용자 참여도
- 직관적이고 사용하기 쉬운 인터페이스

### Weakness (약점)

- 거래 수수료가 없는 구조로 인한 낮은 수익성
- 중고거래 특성상 품질 보증 어려움
- 거래 사기 및 안전 문제 발생 가능

### Opportunity (기회)

- 중고거래 시장 지속 성장
- AI 기반 추천 시스템 도입 가능
- 서비스 확장 및 플랫폼 다각화 가능

### Threat (위협)

- 경쟁 플랫폼 증가
- 법적 규제 가능성
- 경제 상황 변화에 따른 거래 감소

# 🚀 제안한 서비스 기능

## 🐰 안전 거래 장소 추천 시스템

거래 안전성을 높이기 위해 다음과 같은 **공공 장소 기반 거래 추천 시스템**을 제안했습니다.

- 경찰서
- 공원
- 은행
- 공공시설

이를 통해 사용자 간 직거래 시 발생할 수 있는 위험을 줄이고  
플랫폼 신뢰도를 높이는 전략을 설계했습니다.

## 🤖 AI 기반 서비스 기능

다음과 같은 AI 기능 도입을 제안했습니다.

- AI 판매글 자동 생성 기능
- 이미지 기반 상품 검색 기능
